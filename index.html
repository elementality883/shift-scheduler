🗄 Connect to JSONBin (free shared storage)

1
Go to jsonbin.io and Sign Up for a free account (no credit card).
2
In the left sidebar click API Keys → Create Access Key. Name it ShiftScheduler and copy the key.
3
Click Bins → Create Bin. Paste this as the content:
{"team":[],"avail":{}} Name it ShiftScheduler and click Create. Copy the Bin ID from the URL.
4
Paste both values below, set a passphrase, and click Connect. Share the API Key, Bin ID, and passphrase with your team — they only need to do this once.
